About us
Founded in 2011 by Misael. Cleanzone (before Mav anti-malware) is a utility software for android. Cleanzone focuses on realtime protection and power optimizer. We trying to block virus and malware from around the world. We are still developing anti malware for the next generation with machine learning method.

Privacy Policy
Cleanzone is software that works in realtime and requires special access in order to work properly, including
  1. Permission to access storage to be able to scan entire directories on your devices
  2. permission to access internet to display ads
  3. Permission to run Cleanzone on startup
  4. Permission to delete packages that are considered dangerous
  5. Permission to delete files that are considered junk
  6. Permission to turn off processes that can slow down device performance
History

The development of Cleanzone has been started since 2011. At that time the source code was still built on the CMC (Codenesia Malware Cleaner) which used vb language. The detection method uses md5, crc32, sha1 hashing.

In 2014, the source code for Cleanzone was rewritten from scratch with the same programming language. Improved User Interface and added some features. In the following year development of Cleanzone was stopped.

In 2019 the Cleanzone project was back on track. Cleanzone source code rewritten in C# language. The detection method is replaced by a familiar hash which is expected to be able to detect polymorphic viruses.
