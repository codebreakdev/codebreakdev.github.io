Privacy Policy
Mav Anti-Malware is software that works in realtime and requires special access in order to work properly, including
  1. Permission to access storage to be able to scan entire directories on your devices
  2. permission to access internet to display ads
  3. Permission to run Mav Anti-malware on startup
  4. Permission to delete packages that are considered dangerous
  5. Permission to delete files that are considered junk
  6. Permission to turn off processes that can slow down device performance
History

The development of mav anti-malware has been started since 2011. At that time the source code was still built on the CMC (Codenesia Malware Cleaner) which used vb language. The detection method uses md5, crc32, sha1 hashing.

In 2014, the source code for mav anti-malware was rewritten from scratch with the same programming language. Improved User Interface and added some features. In the following year development of mav anti-malware was stopped.

In 2019 the mav anti-malware project was back on track. Mav anti-malware source code rewritten in C# language. The detection method is replaced by a familiar hash which is expected to be able to detect polymorphic viruses.
